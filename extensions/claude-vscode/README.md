# Claude Assistant VSCode Extension

Extension สำหรับใช้งาน Claude AI ใน VSCode

## การติดตั้ง

1. เปิด VSCode
2. กด `Ctrl+Shift+P` (หรือ `Cmd+Shift+P` บน Mac)
3. พิมพ์ "Extensions: Install from VSIX..."
4. หรือคัดลอกโฟลเดอร์นี้ไปที่ `~/.vscode/extensions/`

## การใช้งาน

### วิธีที่ 1: ใช้ Command Palette
1. กด `Ctrl+Shift+P`
2. พิมพ์ "Chat with Claude"
3. กด Enter

### วิธีที่ 2: ใช้ Keyboard Shortcut
- กด `Ctrl+Shift+C` (Windows/Linux)
- กด `Cmd+Shift+C` (Mac)

## คุณสมบัติ

- แชทกับ Claude AI ผ่าน input box
- แสดงผลใน webview panel
- รองรับภาษาไทย
- ใช้ keyboard shortcut ได้
